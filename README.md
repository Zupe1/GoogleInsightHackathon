# Hometown Success Engine 🏅

A fan-centric tool exploring how American geography could help find Team USA Olympic and Paralympic talent, powered by Gemini AI and Google Cloud.

## Live App
Visit: https://hometown-success-engine-462080881599.us-central1.run.app/
No login required — fully accessible for judging.

## Tech Stack
- Next.js + Tailwind CSS
- Gemini 2.5 Flash API
- Google Cloud Run
- Public Team USA data + NOAA climate data
